# Alicebot-MD

> بوت واتساب متكامل مبني بـ Node.js و Baileys

[![Node.js](https://img.shields.io/badge/Node.js-18+-green?style=for-the-badge&logo=node.js)](https://nodejs.org)
[![Baileys](https://img.shields.io/badge/Baileys-Pro-blue?style=for-the-badge)](https://github.com/WhiskeySockets/Baileys)
[![License](https://img.shields.io/badge/License-MIT-purple?style=for-the-badge)](LICENSE)

</div>

---

## ✨ المميزات

- 🤖 ردود تلقائية ذكية
- 🎵 تحميل ميديا من منصات متعددة
- 🎮 ألعاب تفاعلية داخل المجموعات
- 🛡️ نظام حماية للمجموعات
- 👑 نظام صلاحيات متدرج (أونر - بريم - يوزر)
- 📡 تحديثات تلقائية من الـ API
- 🔒 نظام تشفير وحماية للسكريبت
- ⚡ أداء سريع وخفيف على السيرفر

---

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|--------|-----------|
| Node.js | بيئة التشغيل |
| Baileys Pro | مكتبة واتساب |
| Express.js | الـ API الخاص بالبوت |
| LowDB | قاعدة البيانات |
| Chalk | تنسيق الكونسول |
| Moment.js | التاريخ والوقت |
