## 🚀 تشغيل البوت على Termux

```bash
# ══ تحديث وتثبيت المتطلبات ══
pkg update && pkg upgrade -y
pkg install nodejs git -y

# ══ استنساخ البوت ══
git clone https://github.com/fazbear-Official/Alicebot-MD
cd Alicebot-MD

# ══ تثبيت المكتبات وتشغيل البوت ══
npm install
npm start
```

> 💡 عند التشغيل للمرة الأولى اختر طريقة الربط:
> - **`1`** ← QR Code
> - **`2`** ← Pairing Code
